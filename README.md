# 🌐 My First Flask Website

## 📌 Project Description

My First Flask Website is a beginner-friendly full-stack web application built using **Python Flask**, **HTML**, **CSS**, and **JavaScript**. This project demonstrates how to create a simple website using the Flask framework, serve HTML templates, apply CSS styling, and add JavaScript functionality.

It is an excellent project for beginners who are learning Flask and web development.

---

## 🚀 Features

* Home page built with Flask
* Responsive HTML page
* External CSS styling
* JavaScript button interaction
* Flask template rendering
* Beginner-friendly project structure

---

## 🛠️ Technologies Used

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/my-first-flask-website.git
```

2. Open the project folder:

```bash
cd my-first-flask-website
```

3. Install Flask:

```bash
pip install flask
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```
